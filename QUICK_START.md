# 🚀 Quick Start - NeuroQuest Academic Chatbot

## 3-MINUTE SETUP

### Step 1: Open Training Dashboard
```
📁 Open: training-dashboard.html
```

### Step 2: Test Existing Knowledge
Try these questions in the test chat:
- "what is photosynthesis"
- "what is 2 plus 2"
- "what is noun"
- "what is variable"

### Step 3: Start Training!
1. See "Unanswered Questions" section
2. Click "Train Now" on any question
3. Add your answer
4. Click "Add to Knowledge Base"
5. Done! ✅

## 📚 ADD YOUR FIRST ANSWER

**Example:**

```
Subject: Science 🔬
Level: Class 6-8 📘

Question: what is mitosis

Answer: 
Mitosis is cell division! 🔬

1 cell splits into 2 identical cells.

Steps:
1. Prophase
2. Metaphase  
3. Anaphase
4. Telophase

Result: 1 → 2 cells!
```

**Click "Add" → Done!** 🎉

   - Add bullet points

   - Check unanswered questions
   - Add answers weekly
   - Test your chatbot
